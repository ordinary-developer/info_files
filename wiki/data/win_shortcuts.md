# Windows shortcuts

## MMC (Microsoft Management Consoles)
- `devmgmt.msc`  
   device manager MMC
   
- `diskmgmt.msc`  
   disk management MMC
   
- `eventvwr.msc`  
   event viewer MMC
   
- `compmgmt.msc`  
   computer management MMC
   
- `gpedit.msc`  
   group policy MMC
   
- `lusrmgr.msc`  
   local users and groups MMC
   
- `secpol.msc`  
   local security policy MMC
   
- `services.msc`  
   service manager MMC
   
- `wf.msc`  
   windows firewall MMC
