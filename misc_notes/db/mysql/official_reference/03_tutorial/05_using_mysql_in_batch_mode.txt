

 OS shell running
 ================

 to run a script by an anonymous user
 ------------------------------------------------------------------------
 | $ mysql < batch-file							|
 ------------------------------------------------------------------------

 to run under the concrete user
 ------------------------------------------------------------------------
 | $ mysql -h host -u user -p < batch-file				|
 ------------------------------------------------------------------------



 mysql shell running
 ===================
  
 for running under mysql shell your can
 ------------------------------------------------------------------------
 | mysql> SOURCE full_path_to_the_file					|
 ------------------------------------------------------------------------
 
 or 
 ------------------------------------------------------------------------
 | mysql> \. full_path_to_the_file					|
 ------------------------------------------------------------------------
